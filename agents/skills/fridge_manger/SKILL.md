# Skill: Fridge Manager

## Responsibility
Ти відповідаєш за те, щоб дані в `smart_fridge.py` завжди були актуальними.

## Workflow
1. Якщо користувач каже "Що в мене є?", викликай функцію `get_fridge_status`.
2. Якщо термін придатності менше 2 днів, автоматично позначай продукт тегом "URGENT".
3. Координуй дані з модулем `inventory_manager`.

## Constraints
- Не видаляй продукти без підтвердження користувача.
- Завжди перевіряй валідацію через Pydantic-моделі.
